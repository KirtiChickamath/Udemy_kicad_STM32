# STM32 
This project is a simple STM32 based board designed as part of learning and practicing KiCad.It is built around STM32F103C8T6 microcontroller and includes USB input, regulated to 3.3V supply using AMS1117
## Project Details

**Project Name:** STM32 board

**Software Used:** KiCad

**Learning Source** Learn KiCad V6 and STM32 Hardware Design from Udemy
